Magic Square

POST /api/algorithms/numbers/magic-square — checks whether an n×n matrix is a magic square (all rows, columns, and diagonals sum to the same value, using each integer from 1 to n² exactly once).

One row per line. Values separated by spaces or commas. The matrix must be square (n×n), with n between 1 and 100, and contain only positive integers.
