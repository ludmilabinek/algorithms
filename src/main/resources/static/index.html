Algorithms

Classic algorithms exposed as REST endpoints. Pick one to try it out.

Available

Merge sorted lists

Classic merge step from merge sort. Combines two sorted integer lists into one sorted list in linear time.

Time: O(n+m) Space: O(n+m)
POST /api/algorithms/lists/add
Run →

Maximum stock profit

Given daily stock prices, find the maximum profit from a single buy/sell transaction (buy one day, sell a later day). Returns -1 when every possible trade is a loss.

Time: O(n) Space: O(1)
POST /api/algorithms/stocks/max-profit
Run →

RLE compression

Run-Length Encoding: compresses consecutive runs of identical letters into count+char pairs (e.g. "aaabbc" → "3a2b1c"). Optional case-insensitive mode.

Time: O(n) Space: O(n)
POST /api/algorithms/strings/rle-compress
Run →

Minutes between two times

Given two 12-hour clock times separated by a dash (e.g. "1:00pm-11:00am"), returns the total minutes between them. End time wraps to the next day when on or before the start.

Time: O(1) Space: O(1)
POST /api/algorithms/times/minutes-between
